NATS in-container deployment for using in out-of-process tests.

# Installation and usage

Install the package from NuGet:

```shell
dotnet add package Eshva.Testing.OutOfProcessDeployments.Nats
```

Or via the NuGet Package Manager:

```shell
Install-Package Eshva.Testing.OutOfProcessDeployments.Nats
```

Or via appropriate window in Rider or Visual Studio.
